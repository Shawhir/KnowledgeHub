# Naming Conventions



Standard Asset Naming

•	Use descriptive, consistent naming: SubSet\_Feature\_Size\_cColor

o	Subsets: Indicates the sub-material name (e.g., AluminiumPanels\_Brushed)

o	Feature: Focuses on surface feature or style (e.g., Classic, RidgesSmall)

o	Size: Represents measurements in meters (e.g., 1x1, 0.5x0.5)

o	cColor: Color name based on RAL color chart; prefix 'c' indicates colorizable material



Examples

•	AluminiumPanels\_Smooth\_1x2.5\_cBlackBlue

•	CeramicTiles\_RidgesLarge\_0.5x0.25\_cColzaYellow

•	FibreCementPanels\_1x2\_cMouseGrey (No Feature)

•	TravertinePanels\_Smooth\_1x2 (No cColor)

•	ConcreteInSitu\_Boardmarked\_1.66x2 (No cColor)



Note: Refer to Material Directories PDF Pgs 7 \& 8 from Summer for complete reference on naming conventions.
